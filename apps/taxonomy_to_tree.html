Taxonomy to tree

This app will make a tree out of a taxonomy provided as a table in text format. The separator can be provided and the branch length separating taxa in the hierarchy. You may select to have nodes for each taxa or collapse nodes for monotypic taxa (which may be good for downstream analyses).

Leading and trailing spaces in taxon names will be ignored. If several characters are given as separating characters they will be treated as individual characters separating columns, and each occurrence of the character will be taken as separating columns, e.g. many spaces in a row will be taken to separate many columns if space is used as a delimiter of columns. There is no escape of characters used to delimit columns (e.g quotation marks will be interpreted as part of the taxon name and not as text delimiters). \t will be interpreted as a tab.

The app is intended to work similar to the "taxonomy_to_tree.pl" script of Tedersoo et al. 2018 but has been written separately, so there are differences in behaviour. For both applications it is recommended that you get the outcome that you desired.
